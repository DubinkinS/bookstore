
## Bookstore Management API

Проект представляет собой RESTful-приложение для управления книжным магазином.
Система охватывает основные аспекты работы: авторы, книги, заказы, доставка, персонал, должности и магазины.

---

### Используемые технологии

| Технология          | Назначение                                         |
| ------------------- | -------------------------------------------------- |
| Java 21             | Язык программирования                              |
| Spring Boot 3.x     | Основной фреймворк приложения                      |
| Spring Data JPA     | ORM — доступ к базе данных через репозитории       |
| PostgreSQL          | Реляционная база данных                            |
| Lombok              | Упрощение шаблонного кода (геттеры/сеттеры и т.д.) |
| Swagger (SpringDoc) | Автоматическая документация REST API               |

---

### Установка и запуск

#### Предварительные требования:

* Java 21+
* Maven
* PostgreSQL (настройки подключения указаны в `application.yaml`)

#### Сборка проекта:

```bash
mvn clean install
```

#### Запуск:

```bash
mvn spring-boot:run
```

### REST API

Каждая сущность (например, Author, Book, Order, Staff) имеет CRUD-контроллер.
Полная документация доступна через Swagger UI.

#### Примеры доступных эндпоинтов:

| Метод  | URI                | Описание                    |
| ------ | ------------------ | --------------------------- |
| GET    | `/authors`         | Получить всех авторов       |
| POST   | `/authors`         | Добавить нового автора      |
| PUT    | `/authors/{id}`    | Обновить имя автора по ID   |
| DELETE | `/orders/{id}`     | Удалить заказ               |
| GET    | `/shops`           | Получить все магазины       |
